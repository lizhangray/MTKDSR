# MTKDSR

## Multi-Teacher Knowledge Distillation for Super Resolution Image Reconstruction 

## Run

```shell

# test
python main.py --Train False --model_name RFDN --checkpoint MTKDSR.pth
# you can set the test datasets in the main.py too.
```
